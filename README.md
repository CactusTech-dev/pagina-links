# TZEU Tech | Página de Links

Página de links oficial e premium da **TZEU Tech**, startup de tecnologia focada em inovação e desenvolvimento de soluções digitais de alta performance.

Esta página reúne canais oficiais de comunicação, redes sociais e informações sobre os serviços oferecidos.

## 🚀 Tecnologias Utilizadas

- **React 19** - Biblioteca para construção da interface.
- **Vite** - Bundler de desenvolvimento rápido.
- **TypeScript** - Tipagem estática para maior robustez de código.
- **TailwindCSS v4** - Framework de estilo utilitário moderno e rápido.
- **Motion (Framer Motion)** - Micro-interações, efeitos de transição e animações fluidas.
- **HTML5 Canvas** - Animação de brisas marinhas 2D de alta performance em segundo plano (otimizada para dispositivos móveis).

## 🛠️ Como Executar Localmente

### Pré-requisitos
Certifique-se de ter o **Node.js** instalado em sua máquina.

### Executando o Projeto:

1. Instale as dependências necessárias:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento local:
   ```bash
   npm run dev
   ```

3. Abra o navegador no endereço indicado (geralmente `http://localhost:3000`).

## 📦 Build para Produção

Para gerar os arquivos estáticos otimizados para produção na pasta `dist`:
```bash
npm run build
```

## 🌐 Hospedagem & Deploy
Este projeto está configurado para deploy contínuo integrado diretamente com a **Vercel**. Qualquer alteração enviada para a branch `main` no GitHub atualizará a produção de forma automática.
